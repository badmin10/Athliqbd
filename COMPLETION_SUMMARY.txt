================================================================================
                            HOW TO USE THIS CODE
================================================================================

STEP 1: Download
  - Save this folder to your computer

STEP 2: Install dependencies (1 minute)
  $ cd victor-bd
  $ npm install

STEP 3: Initialize database (1 minute)
  $ npx prisma generate
  $ npx prisma migrate dev --name init

STEP 4: Populate sample data (30 seconds)
  $ npm run seed

STEP 5: Start the server (30 seconds)
  $ npm run dev

STEP 6: Test
  - Storefront: http://localhost:3000
  - Admin: http://localhost:3000/admin
  - Credentials: admin@waffled.bd / ChangeMe123!

================================================================================
                           TROUBLESHOOTING
================================================================================

"Module not found" errors
  → Run: npm install

"Cannot find module @prisma/client"
  → Run: npx prisma generate

"PORT 3000 already in use"
  → Run: npm run dev -- -p 3001
  → Or kill process: lsof -ti:3000 | xargs kill -9

Admin login not working
  → Clear browser cookies
  → Check .env has AUTH_SECRET set
  → Verify admin user exists: npx prisma studio

Database locked error
  → Unlikely locally, but on production: migrate to PostgreSQL

Build fails with font errors
  → This is sandbox-specific; will work on your machine with internet
  → On your machine: npm run build will succeed

See SETUP_GUIDE.md for more troubleshooting.
